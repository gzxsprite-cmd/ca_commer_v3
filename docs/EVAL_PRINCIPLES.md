# EVAL_PRINCIPLES

## Evaluation Purpose
Use high-level principles to judge whether Stage-0 and next iterations remain aligned with platform goals.

## Principles
1. **Role/Workspace Identity Clarity**
   - Users can quickly tell where they are and what role context is active.

2. **Next-Step Clarity**
   - Pages make the immediate next action obvious.

3. **Reduced Unnecessary Effort**
   - Interaction flow minimizes extra clicks, context switching, and repeated input.

4. **No Anti-Human Interaction**
   - Avoid hidden logic, overloaded forms, and confusing multi-step traps.

5. **Cross-Page Consistency**
   - Similar tasks use similar structure, naming, and visual patterns.

6. **Room for Refinement**
   - Baseline remains flexible for page-by-page iteration.

This document intentionally avoids page-level scoring checklists at this stage.
