# CA_Commer / CA商务协作平台

## Repository Intro
This repository is the Stage-0 foundation for **CA_Commer** (external temporary name: **CA商务协作平台**). It starts as a **Pages demo** to establish a clear, lightweight platform skeleton for future product development.

## Current Phase
- Phase: **Stage-0 (Pages demo first)**
- Status: **Foundation setup**
- Focus: structure, navigation, and consistency baseline

## Current Goal
Build a practical baseline for a digital platform supporting CA commercial daily operations and tracking/planning work, without over-defining final product details.

## Two-Workspace Concept
1. **Daily Execution Workspace**
   - contract, billing, AM initiation, CA approval/signature, CM archive, billing follow-up
2. **Tracking & Planning Workspace**
   - planning, adjustment, SCP analysis, CA management review
